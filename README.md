# 🌾 Bronowicka Stodoła

Oficjalna strona internetowa projektu Bronowicka Stodoła
https://bronowickastodola.pl

# 📌 O projekcie

Bronowicka Stodoła to strona prezentująca na zamówienie dla lokalnej restauracji.

Projekt ma na celu:

przedstawienie informacji o miejscu i jego działalności
ułatwienie kontaktu i promocję lokalnej inicjatywy

# 🧱 Struktura strony

Projekt składa się z sekcji:

Strona główna – wprowadzenie i ogólny opis
O miejscu – informacje o Bronowickiej Stodole
Galeria – zdjęcia i materiały wizualne
Kontakt – dane kontaktowe

# ⚙️ Technologie

HTML / CSS / JavaScript

Celem projektu jest:

uporządkowana prezentacja treści o Bronowickiej Stodole

wsparcie komunikacji i widoczności inicjatywy online
